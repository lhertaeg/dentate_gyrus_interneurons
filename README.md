# Project description

This repository provides the source code to reproduce figure X from the paper "Soma location of parvalbumin-expressing interneurons defines their contribution to feedforward and feedback inhibition".

## Requirements

This Python code (version 3.10.16) was developed and tested on macOS Monterey Version 12.2.1. The file *environment.yml* contains an exhaustive list of the packages and dependencies for the conda environment used. 

## Installation

Clone this repository:
> git clone https://github.com/lhertaeg/dentate_gyrus_interneurons/

Recreate the conda environment to install the required Python packages/libraries:
> conda env create --name dentate_gyrus --file environment.yml

Install the project:
> pip install -e .

The installation should only take a couple of minutes. If you encounter any issues, you can install the conda environment manually using the list of packages and dependencies specified in *environment.yml*.

## Organization & Reproduction

* **results**: contains data generated by the files in *scripts*
* **scripts**: contains the scripts that generate the simulation data and the figures. First, run the file "run_\*.py" to generate the data. Please note that you will have to set flags in the file to either re-run the simulations or use previously generated data. The expected run time varies across the experiments.
* **src**: contains the code that is shared by several scripts
